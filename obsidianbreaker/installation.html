Installation Steps

  1. Download Obsidianbreaker from Spigot or SavageLabs Downloads
  2. Stop your Minecraft Server
  3. Remove any other obsidian breaker plugins and delete the Obsidianbreaker directory if it is present
  4. Place the Obsidianbreaker jar in your plugins directory
  5. Start your Minecraft Server
  6. Check Console to make sure everything starts correctly. If you have any issues report them to us via Discord

Dependencies

Soft Dependencies:

  • PlaceholderAPI
  • WorldGuard
  • Factions
