Installation Steps

  1. Download LeaderboardX from Spigot or SavageLabs Downloads
  2. Stop your Minecraft Server
  3. Remove any other FTOP plugins and delete the FTOP directory if it is present
  4. Place the LeaderboardX jar in your plugins directory
  5. Start your Minecraft Server
  6. Check Console to make sure everything starts correctly. If you have any issues report them to us via Discord

Dependencies

Hard Dependencies:

  • PlaceholderAPI
  • Factions
  • Vault

Soft Dependencies:

  • WildStacker
