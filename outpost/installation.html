Installation Steps

  1. Download OutpostX from Spigot or SavageLabs Downloads
  2. Stop your Minecraft Server
  3. Remove any other oupost plugins and delete the oupost directory if it is present
  4. Place the OutpostX jar in your plugins directory
  5. Start your Minecraft Server
  6. Check Console to make sure everything starts correctly. If you have any issues report them to us via Discord

Dependencies

Hard Dependencies:

  • WorldEdit (or FAWE)

Soft Dependencies:

  • PlaceholderAPI
  • Holographic Displays
  • ShopGUI+
