Steps

  1. Download SavageCollectors from Spigot or SavageLabs Downloads
  2. Stop your Minecraft Server
  3. Remove any other collector plugins and delete the collector directory if it is present
  4. Place the SavageCollectors jar in your plugins directory
  5. Start your Minecraft Server
  6. Check Console to make sure everything starts correctly. If you have any issues report them to us via Discord

Dependencies

The plugin doesn't have any dependencies.
