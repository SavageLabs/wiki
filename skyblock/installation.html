#Installation Steps

  1. Download SkyblockX from Spigot or SavageLabs Downloads
  2. Stop your Minecraft Server
  3. Remove any other skyblock plugins and delete the skyblock directory and any skyblock worlds if they are present
  4. Place the SkyblockX jar in your plugins directory
  5. Start your Minecraft Server
  6. Check Console to make sure everything starts correctly. If you have any issues report them to us via Discord
