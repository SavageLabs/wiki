Notice: To install an addon you must be using FactionsX. You must run the server at least once to generate the FactionsX files and folders and then turn off your server. Put the addon jar's inside that "Addons" folder which is located inside the FactionsX folder and start your server (your server must be off. /fx reload will NOT work). Folders for each addon will then get generated inside the "Addons" folder where your config files for the addons will be located.

FWild-Addon

The FWild-Addon is a simple addon which adds the choice for a player to use "/f wild" or "/f wild-tp" to randomly teleport into the wilderness. When executing the command, it will look for any unclaimed space within the radius and it will preload the chunk you're about to teleport into.
