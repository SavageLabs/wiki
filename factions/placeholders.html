Internal Placeholders

FactionsX has internal placeholders used in basic configuration sections like faction info from /f show and /f who.

PlaceholderAPI

FactionsX hooks into PlaceholderAPI with normal and relational placeholders.

Please Note: You will need PlaceholderAPI for any of the below placeholder to work. FactionsX automatically loads PAPI placeholders on start up so there is no need to download anything from the ecloud for this. Please do check if the plugin you're trying to use the placeholders on supports PAPI placeholders before asking for help on our support discord about placeholders.
