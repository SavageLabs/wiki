About

FactionsX is a modern implementation of the factions gamemode written from the ground up with extensibility, speed, and usability in mind.

The plugin is a  PREMIUM plugin which can be obtained @ https://patreon.com/ProSavage for only $5 (VIP Package) a month. Any package that is VIP or higher gets access to the FactionsX plugin.

FactionsX is modular, in the context that, it supports addons. They can simply be dropped inside of the addons folder generated inside of the FactionsX plugin folder.

There are a couple addons which have been created by SavageLabs. These addons are:

  • FGrace-Addon (VIP or higher package)
  • FPrinter-Addon (MVP or higher package)
  • FTNT-Addon (VIP+ or higher package)
  • FTOP-Addon (VIP+ or higher package)
  • FWILD-Addon (VIP or higher package)

Once again, these addons can be accessed by subscribing to a package @ https://patreon.com/ProSavage. As a developer, you can create your own addon for FactionsX. We are currently working on a cloud for any free addons that developers want to add (a bit like the ecloud for PlaceholderAPI). You can find everything you need to create an addon here.
